# Guardian Frontend

Base frontend setup with React + Vite + TypeScript and PWA support.

## Stack
- React 19
- Vite 8
- TypeScript
- vite-plugin-pwa

## Scripts
- `npm run dev` - run development server
- `npm run build` - create production build
- `npm run preview` - preview production build

## Mobile-first styles
Global styles start in `src/styles/base.css` and use mobile breakpoints with `min-width` rules.

## Architecture
Project architecture and API contract are documented in:

- `docs/FRONTEND_ARCHITECTURE.md`

## PWA notes
- Manifest generated by `vite-plugin-pwa` config in `vite.config.ts`
- Service worker registration in `src/main.tsx`
- Icons in `public/`
